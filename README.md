# better-server

Современный и красивый файловый сервер на Go — улучшенная альтернатива `python3 -m http.server`.

<img width="1920" height="906" alt="image" src="https://github.com/user-attachments/assets/2e34fba8-6540-453e-bdba-eef60e01a247" />


## Особенности

- Красивый современный интерфейс с тёмной темой
- Превью изображений и видео
- Поддержка скрытых файлов (`.dotfiles`) с удобным переключателем
- Адаптивная сетка карточек
- Человекочитаемые размеры файлов
- Сортировка (папки сверху)
- Защита от path traversal
- Без внешних зависимостей

## Установка

Необходимо установить Golang

```bash
go build -o better-server main.go
sudo cp better-server /usr/local/bin/
sudo chmod +x /usr/local/bin/better-server
```

## Использование

### Быстрый запуск

```bash
# Запуск в текущей директории на порту 8080
better-server

# С указанием порта
better-server -p 3000

# С указанием директории
better-server -d /path/to/folder -p 8080
```

### Параметры

| Параметр     | Описание                        | По умолчанию    |
|--------------|----------------------------------|-----------------|
| `-p`         | Порт сервера                     | `8080`          |
| `-d`         | Директория для раздачи           | текущая (`.`)   |

## Примеры

```bash
# Запуск в папке с музыкой
better-server -d ~/Music -p 9000

# Запуск в домашней директории
better-server -d ~ -p 8080
```

После запуска откройте в браузере:  
**http://localhost:8080**
