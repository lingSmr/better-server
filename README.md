# better-server

Современный и красивый файловый сервер на Go — улучшенная альтернатива `python3 -m http.server`.

<img width="1920" height="906" alt="image" src="https://github.com/user-attachments/assets/2e34fba8-6540-453e-bdba-eef60e01a247" />


## Особенности

- **Красивый современный интерфейс** с тёмной темой (Tailwind CSS)
- **Превью** изображений, видео и аудио
- **Встроенный медиаплеер** для видео и аудио с полным управлением:
  - Play/Pause (пробел), перемотка (←/→), громкость, скорость воспроизведения
  - Drag-перемотка по прогресс-бару, полноэкранный режим (F)
  - Автоскрытие элементов управления
  - Поддержка аудиоформатов (MP3, WAV, FLAC, AAC, M4A, WMA, OPUS)
  - Поддержка видеоформатов (MP4, WebM, OGG, MOV, AVI)
- **Скачивание нескольких файлов** одним ZIP-архивом через выбор чекбоксами
- **Поддержка скрытых файлов** (`.dotfiles`) с удобным переключателем (состояние сохраняется в `localStorage`)
- Адаптивная сетка карточек
- Человекочитаемые размеры файлов
- Сортировка (папки сверху)
- Защита от path traversal
- Без внешних зависимостей (только стандартная библиотека Go)

## Установка

Необходимо установить [Golang](https://go.dev/dl/).

```bash
go build -o better-server main.go
sudo cp better-server /usr/local/bin/
sudo chmod +x /usr/local/bin/better-server
```

## Использование

### Быстрый запуск

```bash
# Запуск в текущей директории на порту 8080
better-server

# С указанием порта
better-server -p 3000

# С указанием директории
better-server -d /path/to/folder -p 8080
```

### Параметры

| Параметр | Описание              | По умолчанию  |
|----------|-----------------------|---------------|
| `-p`     | Порт сервера          | `8080`        |
| `-d`     | Директория для раздачи | текущая (`.`) |

### Навигация

- **Чекбоксы** на карточках — выберите несколько файлов и нажмите «Скачать выбранное» для загрузки ZIP-архива
- **Клик по папке** — переход в директорию
- **Клик по медиафайлу** — открытие встроенного плеера
- **Клик по изображению** — просмотр в браузере
- **Кнопка «Скрытые: выкл/вкл»** — показать/скрыть dot-файлы (состояние запоминается)

### Горячие клавиши плеера

| Клавиша        | Действие                 |
|----------------|--------------------------|
| `Пробел`       | Воспроизведение / Пауза  |
| `←` / `→`      | Перемотка на 10 секунд   |
| `F`            | Полноэкранный режим      |
| `M`            | Отключить / Включить звук |

## Примеры

```bash
# Запуск в папке с музыкой
better-server -d ~/Music -p 9000

# Запуск в домашней директории
better-server -d ~ -p 8080
```

После запуска откройте в браузере:
**http://localhost:8080**
